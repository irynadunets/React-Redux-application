# React-Redux-application
In our application, by clicking on the button of the year we must set title.

Setting the header is shows with a simple example how data rotates inside a redux application, and exactly: 
1. The application has received the initial state
2. User clicking the button sent an action (dispatch action)
3. The corresponding reducer has updated part of the application, in agreement with the fact that learned from the action.
 4. The application has changed and now reflects the new state.
